== Description ==

View Published is a plugin that adds a sub menu under “Pages” that shows only those pages that are published. Get to the right pages faster!


== Installation ==

1. Upload `view-published.zip` to the `/wp-content/plugins/` directory or
2. Upload & activate the plugin through the 'Plugins' menu in WordPress
